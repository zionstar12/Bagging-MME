# depCenR (development version)

## depCenR 0.1.0

* Initial release of the **Bagging-MME** framework.
* Core functions: `depTest()`, `depEst()`, `depEst.aug()`.
* Bivariate generalized-gamma marginals with Clayton copula dependence.
* Parallel computation via `foreach` / `doParallel`.
* Bootstrap confidence intervals for Kendall's τ.
* Placeholder diffusion-model augmentation in `depEst.aug()`.
